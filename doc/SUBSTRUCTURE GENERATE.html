*SUBSTRUCTURE GENERATE

Keyword type: step

This procedure is used to generate a substructure and store the stiffnessmatrix using the *SUBSTRUCTURE MATRIX OUTPUT keyword card. No loading should be applied.

There is one optional parameter: SOLVER. It determines thepackage used to solve the ensuing system of equations. The following solverscan be selected:

  • PARDISO
  • SPOOLES [3,4].

Default is the first solver which has been installed of the following list:PARDISO, SPOOLES. If none is installed, a substructure generation is notpossible.


First line:

  • *SUBSTRUCTURE GENERATE
  • Enter SOLVER, if needed, and its value.

Example:

*SUBSTRUCTURE GENERATE,SOLVER=PARDISO

defines a substructure generation step and selects the PARDISO solver aslinear equation solver in the step. For this to work, the PARDISO solver musthave been linked with CalculiX.
