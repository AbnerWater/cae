*DEPVAR

Keyword type: model definition, material

This keyword is used to define the number of internal state variablesfor a user-defined material. They are initialized to zero at the startof the calculation and can be used within a material usersubroutine. There are no parameters. This card must be preceded by a*USER MATERIAL card.


First line:

  • *DEPVAR
Second line:
  • Number of internal state variables.

Example:

*DEPVAR
12

defines 12 internal state variables for the user-defined material at stake.
